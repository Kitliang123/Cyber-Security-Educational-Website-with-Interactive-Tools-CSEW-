About CSEW

An educational website that shares Cyber Security knowledge.

Cyber Security Educational Website (CSEW) provides as many as possible of the available cyber-attack methods that may be used by the attackers. With that being said, it will not contain any false information as the purpose of the CSEW is not for making money. Not only that, but the CSEW will also include some interactive instructions such as quizzes or step-by-step pictures in order to perform the particular attack. By doing this, specific information or knowledge can be enhanced in the mind of the user for a long time. With that being said, the main objective of this website is to share knowledge related to cyber-attacks as well as solve the problem where many cyber security websites nowadays are boring and just full of words.

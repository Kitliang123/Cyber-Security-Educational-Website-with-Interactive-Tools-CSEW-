John The Ripper

John The Ripper tool is actually a password cracking tool that was released to help IT professionals and common users. The software finds the passwords stored in the local system and also recovers lost passwords. John uses several techniques to crack a password. The use of this tool is complicated for normal users as the tool requires you to have sufficient knowledge of the command line. If you are a skilled IT professional with a strong grip on the command line, John the Ripper is an excellent choice for you. The lack of GUI is the main factor why many users don’t opt for this tool when it comes to cracking Windows passwords.

Features of John The Ripper:

  • John The Ripper is an open source tool.
  • John The Ripper tool is a very simple and easy tool.
  • John The Ripper is a fast password cracker tool.
  • John The Ripper tool can be used on multiple operating systems such as Unix, macOS, Windows, DOS, BeOS, and OpenVMS.
  • John The Ripper tool combines several cracking modes in one program and is fully configurable for your particular needs.

Step-by-step Demonstration of John The Ripper:

    Step 1: Download John The Ripper tool by visiting the website below:
    • https://www.openwall.com/john/
    Step 2: The easiest and usually fastest method is to start John the Ripper and let it automatically step through its most efficient modes. This is usually fastest unless you know the passwords in question will not be on a wordlist.
    • john passwordfile
    Step 3: You can also use a specific wordlist such as one downloaded from the Internet:
    • john passfile –-wordlist=”bestwordlist.txt”
    Step 4: Hopefully at this point you’ve cracked some passwords. You can view any cracked passwords with:
    • john --show passfile
    Step 5: This will show any cracked passwords for that password file. You can also filter the list to show users with a particular id or that are in a particular group:
    • john --show --users=0 passfile
    • john --show --groups=0,1 passfile
    Step 6: Of course you can also pipe the output to another tool such as grep for further filtering:
    • john --show | grep "some user"

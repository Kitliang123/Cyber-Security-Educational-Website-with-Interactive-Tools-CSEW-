Tcpdump

Tcpdump is a command line utility that allows you to capture and analyze network traffic going through your system. It is often used to help troubleshoot network issues, as well as a security tool. A powerful and versatile tool that includes many options and filters, tcpdump can be used in a variety of cases. Since it's a command line tool, it is ideal to run in remote servers or devices for which a GUI is not available, to collect data that can be analyzed later. It can also be launched in the background or as a scheduled job using tools like cron. Tcpdump is native to Linux and does not run on Windows systems. Tcpdump is installed by default on a large number of Linux/UNIX systems. Because tcpdump is so widely used, there is a wealth of support information and articles on the Internet on how to use tcpdump. Features of Tcpdump:

  • Tcpdump is an open source tool.
  • Tcpdump tool is an command-line-based tool.
  • Tcpdump tool is a packet sniffing and packet analyzing tool for a System Administrator.
  • Tcpdump tool allows System Administrator to troubleshoot connectivity issues in Linux.
  • Tcpdump tool is used to capture, filter, and analyze network traffic such as TCP/IP packets going through your system.
  • Tcpdump tool saves the captured information in a pcap file, these pcap files can then be opened through Wireshark or through the command tool itself.

Step-by-step Demonstration of Tcpdump:

    Step 1: To capture the packets of current network interface, use the command below:
    • sudo tcpdump
    Step 2: To capture packets from a specific network interface:
    • sudo tcpdump -i wlo1
    Step 3: To capture specific number of packets, use the following command. This command will capture only 4 packets from the wlo1 interface.
    • sudo tcpdump -c 4 -i wlo1
    Step 4: To print captured packets in ASCII format, use the following command. This command will now print the captured packets from wlo1 to ASCII value.
    • sudo tcpdump -A -i wlo1
    Step 5: To display all available interfaces, use the following command.
    • sudo tcpdump -D
    Step 5: To save captured packets into a file, use the following command.
    • sudo tcpdump -w captured_packets.pcap -i wlo1
    Step 5: To read captured packets from a file, use the following command. This command will now read the captured packets from the captured_packets.pcap file.
    • sudo tcpdump -r captured_packets.pcap
